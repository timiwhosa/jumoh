Course Description

Graphics is a method of communication that we use to portray our ideas to our audience, and the better we can communicate the more information that they can retain. So the graphics helped communicate to be easy, sometimes images are used sometimes icons are used. Its broad and very enjoying.

Duration 2 months

Lessons 3 times weekly

Price N30,000

Required

  1. You must have an Internet connection.

What You will Learn

  • Lesson 1: Getting to know the workspace and fonts/Creating a simple report cover
  • Lesson 2: Using the toolbox/Using color
  • Lesson 3: Drawing and editing objects/Creating a greeting card
  • Lesson 4: Using photographs in a motivational poster
  • Lesson 5: Using a template to create a two-page newsletter
  • Lesson 6: Bitmap and vector effects
  • Lesson 7 Letterhead and data merge
  • Lesson 8: Using scripts and macros to create a calendar
