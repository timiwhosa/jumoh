Course Description

Learn how build beautiful Apps form scratch with Flutter and Dart Programming Language. No prior knowledge or experience is required , you will learn the basic of Programming to advanced concepts.

Duration 2 months

Lessons 3 times weekly

Price N30,000

Required

  1. You must have an Internet connection.
  2. You must have working laptop (for flutter at least with 4gb of RAM).

What You will Learn

  • Learning the Dart Programming Language
  • IDE for flutter developers
  • User Interfaces - Flutter Widgets
  • Styles
  • Assets
  • Static User Interface
  • Dynamic User Interface
  • Animation
  • Storage
  • External Libraries For Flutter Developers
  • Behavior Components
  • State Management in Flutter
  • Firebase
